## 1. 商品类型定义

### 1.1 充值商品 (Recharge)

| 属性 | 说明 |
|------|------|
| 类型 | token充值 |
| 额度 | 固定额度（按配置） |
| 有效期 | 1天 ~ 7天（短期） |
| 速率限制 | 中等 |

**特点**：
- 有固定的 token 数量
- 有效期短，到期清零
- 有 API 调用速率限制

### 1.2 VIP订阅 (VIP Subscription)

| 属性 | 说明 |
|------|------|
| 类型 | VIP会员订阅 |
| 额度 | 不限量 |
| 有效期 | 1个月 ~ 1年（长期） |
| 速率限制 | 根据等级区分 |

**VIP 等级定价**：

| 等级 | 价格 | 速率限制 | 特点 |
|------|------|----------|------|
| 基础版 | ¥9.9/月 | 严格（50 RPM） | 低价限速严格 |
| 中级版 | ¥29.9/月 | 中等（200 RPM） | 中等价格限速中等 |
| 高级版 | ¥99.9/月 | 宽松（无限制） | 高级限速宽松 |

## 2. 管理员功能补充

### 2.1 商品管理

需要添加的功能：

| 功能 | 说明 |
|------|------|
| 商品列表 | 显示所有商品（上架/下架状态） |
| 新建商品 | 添加充值商品或VIP套餐 |
| 上架/下架 | 控制商品前端显示 |
| 编辑商品 | 修改价格、额度等 |
| 删除商品 | 物理删除（谨慎） |

**商品表结构更新**：

```go
type Product struct {
    ID              int64
    Name            string    // 商品名称
    ProductType     string    // recharge / vip
    Price           float64   // 价格
    Quota           int64     // 充值额度（仅recharge有效）
    VIPDays         int       // VIP天数（仅vip有效）
    RateLimitRPM    int       // 速率限制（仅vip有效）
    SortOrder       int       // 排序
    IsRecommended   bool      // 是否推荐
    Status          string    // active / inactive
    CreatedAt       time.Time
    UpdatedAt       time.Time
}
```

## 4. 技术说明

### 4.1 速率限制实现

```go
// 速率限制配置
type RateLimitConfig struct {
    RPM  int // requests per minute
    TPM  int // tokens per minute
}

var VIPRateLimits = map[string]RateLimitConfig{
    "basic":     {RPM: 50,  TPM: 10000},   // 基础版：严格
    "standard":  {RPM: 200, TPM: 50000},   // 中级版：中等
    "premium":   {RPM: 0,   TPM: 0},       // 高级版：无限制
}
```

### 4.2 VIP状态判断

```go
func (u *User) IsVIP() bool {
    if u.VIPExpiredAt == nil {
        return false
    }
    return u.VIPExpiredAt.After(time.Now())
}
```
